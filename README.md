# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## Tools Used
- SQLite
- DB Browser for SQLite

## Dataset
Online Sales Data.csv

## SQL Concepts Applied
- GROUP BY
- ORDER BY
- SUM()
- COUNT(DISTINCT)
- Aggregate Functions
- Date Functions (strftime)

## Analysis Performed
1. Monthly Revenue and Order Volume Analysis
2. Top 3 Months by Revenue
3. Revenue by Region
4. Orders by Payment Method
5. Product Category Performance

## Key Findings
- Monthly revenue trends were analyzed from January to August 2024.
- Order volume was calculated using distinct transaction IDs.
- Revenue contribution was compared across regions.
- Customer payment preferences were identified.
- Product categories were ranked based on revenue generated.
